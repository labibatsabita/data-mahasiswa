## Deskripsi Program

Aplikasi Data Mahasiswa berbasis Node.js dan Express yang digunakan untuk mengelola data mahasiswa. Data disimpan dalam file JSON dan dapat ditambah, diubah, ditampilkan, serta dihapus melalui API.

## Fitur

- Menampilkan data mahasiswa
- Menambahkan data mahasiswa
- Mengubah data mahasiswa
- Menghapus data mahasiswa
- Penyimpanan data menggunakan JSON

## Teknologi yang Digunakan

- Node.js
- Express.js
- JSON
- HTML
- CSS
- JavaScript

## Cara Menjalankan

1. Clone repository

```bash
git clone https://github.com/username/data-mahasiswa.git
```

2. Install dependency

```bash
npm install
```

3. Jalankan aplikasi

```bash
node app.js
```

4. Buka browser

```text
http://localhost:3000
```
